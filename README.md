# JavaScript Concepts Repository

This repository contains implementations and explanations of important **JavaScript fundamentals, advanced concepts, asynchronous programming, utilities, and polyfills**.
The goal of this project is to understand how JavaScript works internally by implementing concepts with practical examples.

---

## Topics Covered

### 01. Basics

Primitive vs non-primitive data types, variable declarations (`var`, `let`, `const`), and differences between `undefined`, `null`, and not-defined.

### 02. Scope

Global, function, and block scope, scope chaining, and variable shadowing.

### 03. Hoisting

How JavaScript hoists variables and functions and the concept of the Temporal Dead Zone (TDZ).

### 04. Functions

Function declaration, expression, anonymous functions, arrow functions, higher-order functions, and IIFE.

### 05. Execution Context

How JavaScript executes code using execution contexts and the call stack.

### 06. Closures

Understanding closures and how functions retain access to their lexical scope.

### 07. Prototypes

Prototype basics, prototype chain, and constructor-based inheritance.

### 08. Object Concepts

Pass-by-value vs pass-by-reference, object creation methods, and deep vs shallow copy.

### 09. Advanced Functions

Currying, infinite currying, memoization, rest parameters, spread operator, and `call`, `apply`, `bind`.

### 10. Async JavaScript

Callbacks, callback hell, promises, `then/catch`, `async/await`, event loop, callback queue, and microtask queue.

### 11. Event System

Event bubbling, event capturing, event delegation, and stopping event propagation.

### 12. JavaScript Internals

Equality comparison, garbage collection, strict mode, lexical environment, and type coercion vs conversion.

### 13. Utility Functions

Performance utilities such as debounce, throttle, map limit, and parallel limit.

### 14. Polyfills

Custom implementations of built-in JavaScript methods like `call`, `apply`, `bind`, array methods, promise utilities, and event emitter.

### 15. Additional Concepts

Generator functions, higher-order functions, prototype vs `__proto__`, variable shadowing, and JavaScript parsing/compilation.
